# 🧮 Calculator using JavaScript

A modern and responsive Calculator built using HTML, CSS, and JavaScript.  
This project performs basic arithmetic operations with a clean user interface, keyboard support, and responsive design.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 📊 Percentage Calculation
- ⌫ Delete Last Character
- 🧹 Clear Screen
- ⌨ Keyboard Support
- 📱 Responsive Design
- ✨ Modern UI with Hover Effects
- ⏰ Real-Time Clock Display

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
